# 02 - 概率统计与优化：SLAM 的不确定性语言

---

## 2.1 为什么 SLAM 需要概率？

SLAM 中的所有测量都有**噪声**：

```
相机测量像素位置 → 误差 1-2 像素
PnP 估计位姿 → 误差几毫米到几厘米
三角化 3D 点 → 深度方向误差大
```

概率论就是描述和处理这些不确定性的工具。

---

## 2.2 基础概念 🔴

### 2.2.1 随机变量

一个不确定的值。例如"相机的真实位置"就是一个随机变量——我们不知道精确值，但知道它在某个范围内的可能性。

### 2.2.2 期望 E[X]

期望是随机变量的**平均值**（加权平均）。

```
E[X] = Σ x_i · P(x_i)          离散情况
E[X] = ∫ x · f(x) dx           连续情况
```

**在 SLAM 中：** 期望 = 我们估计的位置（最优估计）。

### 2.2.3 方差 Var[X]

方差衡量随机变量**偏离期望的程度**。

```
Var[X] = E[(X - E[X])²]

标准差 σ = √Var[X]（和原始数据同单位，更好理解）
```

**在 SLAM 中：** 方差 = 我们对自己估计的**信心**。

```
方差小 → 很确定位姿在哪
方差大 → 不确定位姿在哪
```

### 2.2.4 协方差 Cov[X, Y] 🔴

两个随机变量**一起变化**的程度。

```
Cov[X, Y] = E[(X - E[X])(Y - E[Y])]
```

**直觉：**

```
Cov > 0：X 增大时 Y 也增大（正相关）
Cov < 0：X 增大时 Y 减小（负相关）
Cov ≈ 0：X 和 Y 没有线性关系
```

**⚠️ 这在 SLAM 中极其重要：**

相机的 $x$ 和 $y$ 方向误差不是独立的——它们会互相影响。协方差矩阵就描述这种关系。

---

## 2.3 协方差矩阵 🔴

### 2.3.1 定义

把多个变量之间的协方差排成矩阵：

$$
\Sigma = \begin{bmatrix}
\text{Var}(X_1) & \text{Cov}(X_1,X_2) & \cdots \\
\text{Cov}(X_2,X_1) & \text{Var}(X_2) & \cdots \\
\vdots & \vdots & \ddots
\end{bmatrix}
$$

```python
# 相机位姿 (x, y, θ) 的协方差矩阵
Sigma = np.array([
    [0.1,  0.02, 0.01],    # x 的不确定性 0.1 m²
    [0.02, 0.1,  0.01],    # y 的不确定性 0.1 m²
    [0.01, 0.01, 0.05]     # θ 的不确定性
])
```

### 2.3.2 💡 几何意义

```
协方差矩阵描述了数据的"椭圆"形状：
  对角线 = 每个方向上的展宽
  非对角线 = 椭圆的方向（倾斜）
```

### 2.3.3 信息矩阵

信息矩阵 $\Omega = \Sigma^{-1}$（协方差矩阵的逆）。

```
协方差大 → 信息小 → 这个测量"不可信"
协方差小 → 信息大 → 这个测量"可信"
```

在 BA 中，每个观测的权重 = 它的信息矩阵。

## 2.4 正态分布 🔴

### 2.4.1 一维正态分布

$$X \sim \mathcal{N}(\mu, \sigma^2)$$

- $\mu$：均值（分布的中心）
- $\sigma$：标准差（分布的宽度）

```python
from scipy.stats import norm
x = np.linspace(-5, 5, 100)
y = norm.pdf(x, loc=0, scale=1)  # N(0,1)
plt.plot(x, y)
```

### 2.4.2 多维正态分布

$$X \sim \mathcal{N}(\mu, \Sigma)$$

- $\mu$：均值向量
- $\Sigma$：协方差矩阵

```python
mean = [0, 0]
cov = [[1, 0.5], [0.5, 2]]
data = np.random.multivariate_normal(mean, cov, 1000)
```

### 2.4.3 💡 为什么 SLAM 假设正态分布？

1. **简单**：只需要均值和协方差就能描述整个分布
2. **合理**：大多数传感器噪声确实是正态分布
3. **计算方便**：正态分布经过线性变换还是正态分布

**SLAM 中的所有概率都是用正态分布近似的。**

## 2.5 最大似然估计 MLE 🟡

### 2.5.1 核心思想

**找到参数 $\theta$，让观测到当前数据的概率最大。**

```python
# 假设观测服从 N(μ, σ²)
# 已知 σ=1，观测数据为 [2.1, 1.9, 2.0, 2.2, 1.8]
# 求 μ 的 MLE 估计

data = np.array([2.1, 1.9, 2.0, 2.2, 1.8])
mu_mle = data.mean()  # 均值就是 MLE 估计！
```

### 2.5.2 在 SLAM 中

**最小化重投影误差 = 最大化观测的似然。**

```
BA 的目标函数：
min Σ (重投影误差)²

等价于（假设正态噪声）：
max P(观测 | 位姿, 3D点)
```

这就是：**最小二乘 = 最大似然**（在高斯噪声假设下）。

## 2.6 贝叶斯公式 🔴

### 2.6.1 公式

$$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$$

| 项 | 名称 | 含义 |
|----|------|------|
| $P(A\|B)$ | 后验 | 看到数据后，对 A 的信念 |
| $P(B\|A)$ | 似然 | 假设 A 正确时，看到 B 的概率 |
| $P(A)$ | 先验 | 看数据前，对 A 的信念 |
| $P(B)$ | 证据 | 不太重要（归一化常数） |

### 2.6.2 💡 SLAM 中怎么用

```
P(位姿 | 观测) = P(观测 | 位姿) × P(位姿) / P(观测)

更新前                   更新后
  │                       │
  ▼                       ▼
先验分布  ×  似然  =  后验分布
（预测的） （测量）  （修正后的）
```

**这就是卡尔曼滤波的核心思想！**

## 2.7 非线性优化 🔴

### 2.7.1 SLAM = 一个巨大的优化问题

```
目标：找到所有相机位姿和 3D 点
      让所有重投影误差的平方和最小
       
这不是一次能算出来的 → 需要迭代优化
```

### 2.7.2 梯度下降法

```
x_{k+1} = x_k - α × ∇f(x_k)

每次沿着"最陡"的方向走一小步
```

**问题：** 太慢，且 α 难调。

### 2.7.3 高斯-牛顿法

专门为**最小二乘** $f(x) = \frac{1}{2}\|r(x)\|^2$ 设计：

$$(J^T J) \Delta x = -J^T r$$

其中 $J$ 是雅可比矩阵（导数矩阵）。

```python
def gauss_newton_step(J, residuals):
    """一次高斯-牛顿迭代"""
    delta = np.linalg.solve(J.T @ J, -J.T @ residuals)
    return delta
```

### 2.7.4 LM 算法（最常用）

**高斯-牛顿 + 梯度下降 = LM**

$$(J^T J + \lambda I) \Delta x = -J^T r$$

```
λ 大 → 接近梯度下降（慢但稳定）
λ 小 → 接近高斯-牛顿（快但可能发散）
LM 自动调节 λ → 又快又稳
```

**SLAM 中的 BA 就是用 LM 实现的。**

---

## 2.8 雅可比矩阵 🔴

### 2.8.1 什么是雅可比

雅可比 $J$ 是一个**导数矩阵**：

$$
J_{ij} = \frac{\partial f_i}{\partial x_j}
$$

第 $i$ 行第 $j$ 列 = 第 $i$ 个输出对第 $j$ 个输入的偏导。

### 2.8.2 例子

```python
# f(x, y) = [x² + y, x * y]
# J = [[∂f₁/∂x, ∂f₁/∂y],
#      [∂f₂/∂x, ∂f₂/∂y]]
#   = [[  2x,    1 ],
#      [   y,    x ]]
```

### 2.8.3 在 SLAM 中

```
重投影函数：π(R, t, X) → (u, v)

雅可比 ∂π/∂R：相机旋转变化时，像素怎么变
雅可比 ∂π/∂X：3D 点移动时，像素怎么变
```

BA 中需要这两个雅可比来指导优化。
