## 1.1 向量：SLAM 里最基础的数据结构

### 1.1.1 什么是向量

向量就是一列数。在 SLAM 中：

```
[1.2, 3.5, 0.8]  ← 一个 3D 点的坐标 (X, Y, Z)
[0.5, 2.1]        ← 一个特征点的像素坐标 (u, v)
[-0.3, 0.1, 0.9]  ← 相机的朝向（旋转向量的一部分）
```

### 1.1.2 基本操作

```
加法：[1,2] + [3,4] = [4,6]           → 平移两个点
数乘：2 × [1,2,3] = [2,4,6]           → 缩放一个向量
点积：[1,2]·[3,4] = 1×3 + 2×4 = 11   → 判断方向是否一致
叉积：[1,0,0]×[0,1,0] = [0,0,1]      → 求垂直于两个向量的方向
```

```python
import numpy as np
a = np.array([1.0, 2.0, 3.0])
b = np.array([4.0, 5.0, 6.0])
print(a + b)                     # 加法
print(np.dot(a, b))              # 点积
print(np.linalg.norm(a))         # 模长（长度）
```

### 1.1.3 💡 为什么点积这么重要？

```
点积 = 0 → 两个向量垂直 → 两个方向无关
点积 > 0 → 指向大致相同
点积 < 0 → 指向大致相反

在 SLAM 中：
- 判断特征点是否在相机前方（视线方向与特征方向点积 > 0）
- 判断相机是否在朝目标移动
```

## 1.2 矩阵：SLAM 的计算核心

### 1.2.1 什么是矩阵

矩阵就是排成矩形的一组数。3×3 矩阵 = 3 行 3 列。

### 1.2.2 SLAM 中出现的矩阵

```
2×2 协方差矩阵：    特征点的不确定性
3×3 旋转矩阵：      相机的朝向
3×3 内参矩阵 K：    相机的焦距、中心
3×3 本质矩阵 E：    两张图之间的几何关系
3×4 投影矩阵 P：    从 3D 到 2D 的投影
4×4 变换矩阵 T：    旋转 + 平移（SLAM 里最常用的）
```

### 1.2.3 矩阵乘法

```python
A = np.array([[1, 2], [3, 4]])   # 2×2
B = np.array([[5, 6], [7, 8]])   # 2×2
C = A @ B                         # 矩阵乘法
```

**规则：** $A_{m\times n} \times B_{n\times p} = C_{m\times p}$

**即：** 左边矩阵的列数 = 右边矩阵的行数。

```python
# 常见形状
R = np.eye(3)          # 3×3 旋转矩阵
P = np.array([1,2,3])  # (3,) 3D点
P_rotated = R @ P      # (3,) 应用旋转 → 3D点绕原点转动
```

## 1.3 单位矩阵与逆矩阵 🔴

### 1.3.1 单位矩阵 I

对角线是 1，其他是 0。任何矩阵乘 I 等于自己。

```python
I = np.eye(3)      # 创建 3×3 单位矩阵
```

**在 SLAM 中的意义：单位矩阵 = "没有旋转"、"没有变换"。**

### 1.3.2 逆矩阵

$A \times A^{-1} = I$

```python
A = np.array([[1, 2], [3, 4]])
A_inv = np.linalg.inv(A)
print(A @ A_inv)  # 应该接近单位矩阵
```

**💡 逆矩阵在 SLAM 中 = "反向变换"：**

```
T = 世界→相机的变换
T⁻¹ = 相机→世界的变换

P_cam = T @ P_world     # 世界点 → 相机坐标
P_world = T⁻¹ @ P_cam   # 相机坐标 → 世界点
```

这是 SLAM 中**最频繁**的操作之一。

## 1.4 旋转矩阵 🔴

### 1.4.1 什么是旋转矩阵

旋转矩阵 $R$ 是 3×3 的矩阵，满足：
- $R^T R = I$（正交矩阵）
- $\det(R) = 1$

```python
# 绕 Z 轴旋转 θ 角度
theta = np.pi / 4  # 45度
R_z = np.array([
    [np.cos(theta), -np.sin(theta), 0],
    [np.sin(theta),  np.cos(theta), 0],
    [0,              0,             1]
])
```

### 1.4.2 💡 直观理解

```
旋转矩阵的每一列 = 相机坐标轴在世界坐标中的方向

R = [  x_轴方向  |  y_轴方向  |  z_轴方向  ]
        ↓            ↓            ↓
      第1列         第2列         第3列
```

### 1.4.3 旋转矩阵的性质

```python
R = np.array([...])              # 一个旋转矩阵
print(R.T @ R)                   # 应该是单位矩阵 (验证)
print(np.linalg.det(R))          # 应该是 1.0 (验证)

# 反向旋转就是转置
R_inv = R.T                      # R⁻¹ = Rᵀ ！旋转矩阵的逆就是转置
```

## 1.5 变换矩阵 🔴

### 1.5.1 齐次坐标

用 4 维向量表示 3D 点：

```
3D 点 (X, Y, Z) → 齐次坐标 (X, Y, Z, 1)
3D 方向 (X, Y, Z) → 齐次坐标 (X, Y, Z, 0)
```

最后一位是 1 表示点，是 0 表示方向。

### 1.5.2 4×4 变换矩阵

```
T = [ R₃ₓ₃  |  t₃ₓ₁ ]
    [ 0 0 0  |   1   ]
    
    旋转部分    平移部分
```

```python
def build_transform(R, t):
    """R: 3×3, t: (3,), 返回 4×4 变换矩阵"""
    T = np.eye(4)
    T[:3, :3] = R
    T[:3, 3] = t
    return T

# 用变换矩阵变换一个 3D 点
def transform_point(T, p_world):
    """T: 4×4, p_world: (3,), 返回 (3,) 相机坐标"""
    p_h = np.append(p_world, 1.0)     # (X, Y, Z) → (X, Y, Z, 1)
    p_cam_h = T @ p_h                  # 4×4 @ 4×1 = 4×1
    return p_cam_h[:3]                 # (X', Y', Z')
```

### 1.5.3 💡 为什么要用 4×4？

因为 3×3 矩阵只能做旋转，不能做平移。用 4×4 可以把**旋转和平移合在一起**，一次矩阵乘法就完成。

## 1.6 线性方程组与最小二乘 🔴

### 1.6.1 问题

已知一些 3D 点和它们在图像上的 2D 位置，求相机位姿。

这就是 $Ax = b$，但通常 $A$ 不是方阵（方程比未知数多），没有精确解。

### 1.6.2 解法：最小二乘

找一个 $x$ 让误差 $\|Ax - b\|^2$ 最小。

```python
# 用 numpy 解最小二乘
A = np.array([[...], ...])  # m×n 矩阵，m > n
b = np.array([...])          # m 维向量
x, residuals, rank, s = np.linalg.lstsq(A, b, rcond=None)
```

### 1.6.3 在 SLAM 中

PNP、三角化、BA 本质上都是最小二乘问题。

## 1.7 SVD：SLAM 中最重要的矩阵分解 🔴

### 1.7.1 定义

任何矩阵 $A$ 都可以分解为：

$$A_{m\times n} = U_{m\times m} \Sigma_{m\times n} V^T_{n\times n}$$

其中 $U$ 和 $V$ 是正交矩阵，$\Sigma$ 是对角矩阵（对角线上的值叫**奇异值**）。

```python
U, S, Vt = np.linalg.svd(A)
```

### 1.7.2 💡 核心直觉

```
SVD 把矩阵分解成三个"指令"：
  V^T：先旋转
  Σ：   再缩放
  U：   再旋转
```

### 1.7.3 在 SLAM 中的三大用途

**用途 1：解 $Ax=0$（求零空间）**

解 $Ax = 0$ 的**非零解**（即齐次线性方程组）。

**方法：** 对 $A$ 做 SVD，解就是 $V$ 的**最后一列**。

```python
# 解 Ax = 0
U, S, Vt = np.linalg.svd(A)
x = Vt[-1]  # 解的V的最后一列
```

**⚠️ SLAM 中大量使用这个技巧：**

| 问题 | 矩阵 A | 解 x |
|------|--------|------|
| 八点法求本质矩阵 | 匹配点构成的矩阵 | 本质矩阵 E |
| 三角化 | 两帧投影矩阵 | 3D 点的齐次坐标 |
| 单应矩阵估计 | 匹配点构成的矩阵 | 单应矩阵 H |
| PnP (DLT) | 3D-2D 对应 | 投影矩阵 P |

**用途 2：求伪逆**

```
A⁺ = V Σ⁺ Uᵀ  （Σ⁺ 是奇异值的倒数）
```

用于解最小二乘问题，比正规方程数值更稳定。

**用途 3：本质矩阵分解**

从本质矩阵 $E$ 恢复相机运动 $R$ 和 $t$，需要对 $E$ 做 SVD。

```python
U, S, Vt = np.linalg.svd(E)
# 有两组可能的解
W = np.array([[0, -1, 0], [1, 0, 0], [0, 0, 1]])
R1 = U @ W @ Vt
R2 = U @ W.T @ Vt
t1 = U[:, 2]
t2 = -U[:, 2]
# 然后选择 3D 点都在相机前方的那组解
```

## 1.8 特征值与特征向量 🟡

### 1.8.1 定义

对矩阵 $A$，如果存在非零向量 $v$ 满足：

$$Av = \lambda v$$

则 $\lambda$ 是特征值，$v$ 是特征向量。

### 1.8.2 💡 几何意义

```
特征向量 = 矩阵变换的"主轴方向"
特征值   = 那个方向的"缩放倍数"

想象一个圆经过矩阵 A 变换后变成椭圆：
  椭圆的长轴方向 = 特征向量
  长轴/短轴比例 = 特征值之比
```

### 1.8.3 在 SLAM 中

**协方差矩阵的特征分解 = 数据的主轴方向**

```python
Sigma = np.array([[2.0, 0.5], [0.5, 1.0]])
eigvals, eigvecs = np.linalg.eig(Sigma)
print("不确定性最大的方向:", eigvecs[:, 0])
print("该方向的标准差:", np.sqrt(eigvals[0]))
```

这告诉你：相机位置在哪个方向最不确定。
